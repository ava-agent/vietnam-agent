# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-03-03

### Added
- 🎒 **必备物品模块** - 6大分类，40+物品清单，支持勾选持久化
- 💰 **金钱换算模块** - 人民币/越南盾实时换算，物价参考，换钱攻略
- 📱 **常用软件模块** - 12款精选App推荐，分类筛选，一键下载
- 🏖️ **景点攻略模块** - 5大热门城市，景点/美食/交通/预算全覆盖
- 🎨 越南风格主题系统 (颜色/字体/间距/阴影)
- 🔧 ErrorBoundary 错误边界组件
- ⏳ Loading 加载状态组件
- ♿ 无障碍支持 (accessibilityRole/accessibilityLabel)
- 🧪 33个Jest单元测试
- 🚀 GitHub Actions CI/CD 自动构建发布

### Technical
- React Native 0.84.0
- TypeScript 5.8+
- React Navigation 7.x
